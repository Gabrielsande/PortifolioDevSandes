# 🌐 Portfólio – Gabriel Sandes

Este repositório contém o **portfólio pessoal** de Gabriel Sandes, desenvolvido com foco em apresentar suas habilidades, projetos e trajetória profissional como desenvolvedor full-stack em formação.

---

## ✨ Visão Geral

O site foi construído utilizando **HTML**, **CSS modularizado** e **JavaScript puro**, com foco em responsividade, acessibilidade e identidade visual própria.

Ele é dividido em seções intuitivas como:

- Início
- Sobre mim
- Habilidades
- Projetos
- Contato

---

## 📂 Estrutura do Projeto
```
PORTIFOLIODEVSANDES/
├── assets/ # Imagens e ícones usados no site
├── css/
│ ├── components/ # CSS dividido por seções
│ ├── extends.css # Arquivo principal que importa todos os estilos
│ ├── globals.css # Estilos globais (resets, tipografia, utilitários)
│ └── variables.css # Variáveis de cores, fontes e temas
├── scripts/
│ └── index.js # Comportamento da navbar e tema escuro
└── index.html # Página principal do portfólio
```

---

## 💻 Tecnologias Utilizadas

- **HTML5** – Estrutura semântica da página
- **CSS3 (modularizado)** – Estilização responsiva e temas
- **JavaScript (Vanilla JS)** – Menu mobile e alternância de tema
- **Figma** (design visual - mencionado no portfólio)
- **Photoshop & Illustrator** (como skills de design)

## 🎯 Funcionalidades

- ✅ Menu responsivo com botão hambúrguer
- ✅ Modo claro/escuro com `classList.toggle()`
- ✅ Layout 100% responsivo para todos os dispositivos
- ✅ Estrutura em seções com navegação suave (`scroll-behavior`)
- ✅ Arquivo de currículo disponível para download
- ✅ Projeto modularizado com CSS dividido por componentes
